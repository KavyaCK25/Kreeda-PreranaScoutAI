# 🏆 Kreeda-Prerana Scout AI

An Android application built using Kotlin to help identify and track young sports talent in rural schools. The app records physical performance metrics like sprint timing, jump height, and other athletic data to help scouts and coaches discover hidden talent.

---

## 📱 Project Overview

**Kreeda-Prerana Scout AI** is a grassroots sports talent tracking system designed to bridge the gap between rural athletes and professional scouting opportunities. It maintains a digital performance history of students, making talent identification easier and more structured.

---

## 🎯 Problem Statement

Talented young athletes in rural schools often go unnoticed due to the lack of structured performance tracking systems. Coaches cannot maintain long-term data, making it difficult to identify consistent performers.

---

## 🚀 Features

- Student athlete profile management  
- Performance tracking (running time, jump height, etc.)  
- Historical performance records  
- Easy-to-use Android UI  
- Offline data storage support
